Keep those [bones](https://github.com/ghostmechanics/innards) squeaky clean.

![clean skeleton](clean_skeleton.gif)

```shell
npm i -D eslint-config-innards
```

Now add this to your `.eslintrc.js`:

```js
'use strict';

module.exports = {
    extends: 'innards'
};
```
